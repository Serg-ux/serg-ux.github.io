## Reconnaissance

A full TCP port scan was run against the target to identify the exposed services and their versions.

```bash
nmap -sVC -p- --open -sS --min-rate 5000 -v -n --stats-every=5s -Pn -oN sense_scan <IP>
```

Two ports are open: port 80 (HTTP) and port 443 (HTTPS), both served by **lighttpd 1.4.35**. Browsing the application reveals a login page on the web server.

![Login page on the web server](/assets/images/sense/138-1.png)
*Login page exposed by the web server*

The interface and response headers make it clear that this is a **pfSense** firewall web interface.

## Enumeration / Analysis

The application might look like a candidate for client-side issues such as XSS, but the more productive path is directory enumeration to find misconfigurations or exposed files.

```bash
gobuster dir -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -u http://<IP> -b 301 -x txt,php,html
```

As a more exhaustive alternative, DirBuster can enumerate over HTTPS.

```bash
dirbuster -u https://10.10.10.60 -t 20 -l /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -r sense-10.10.10.60/dirbuster_dir-med -e php,txt,html
```

Enumeration surfaces several interesting paths, one of which exposes sensitive information.

![Sensitive path discovered](/assets/images/sense/138-2.png)
*An enumerated path exposing sensitive data*

Among those files are credentials stored in plaintext.

![Plaintext credentials](/assets/images/sense/138-3.png)
*Credentials stored in cleartext*

These credentials authenticate successfully against the pfSense dashboard.

```bash
rohit:pfsense
```

## Exploitation

With administrative access to the pfSense interface, the next step is to find a known vulnerability that yields remote code execution. The graphing functionality is reachable, and it can be abused through a dedicated Metasploit module.

The module **exploit/unix/http/pfsense_graph_injection_exec** fits this case, since it injects commands through the graphing feature. Launching it returns a Meterpreter session on the target.

![Meterpreter session obtained](/assets/images/sense/138-4.png)
*Meterpreter session through the pfSense graphing injection*

## Privilege Escalation

The Meterpreter session already runs with enough privileges to read both the user and root flags, so the machine is fully compromised.
