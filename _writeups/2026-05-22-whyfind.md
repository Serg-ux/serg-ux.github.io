### Extracting Network Information with RegRipper

To extract information about the network lease (the temporary IP address assigned by DHCP), connection times, and related details, **RegRipper** is used to dump the relevant registry keys.

> RegRipper is a registry parsing tool that applies a set of known plugins to extract structured information — such as network history, DHCP lease data, and connection timestamps — from raw registry hives like SYSTEM and NTUSER.DAT.

```bash
rip.pl -r SYSTEM -p networklist
```

![RegRipper output showing network connection details](/assets/images/whyfind/Pasted-image-20260522163924.png)
*Network connection details extracted via RegRipper*
