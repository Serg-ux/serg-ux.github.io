Photon Lockdown

Reconnaissance

The analysis began by extracting the provided archive. After unzipping the file, three separate files were identified.

I inspected the file named fwu_ver using cat, which revealed the firmware version in use. The version identified was 3.0.5.

Enumeration / Analysis

Next, I analyzed the rootfs file to determine its format. Using the file command, it was identified as a SquashFS filesystem, little endian, version 4.0, compressed with zlib.

Given this information, I proceeded to extract the filesystem using unsquashfs. 

unsquashfs rootfs

Once the filesystem was extracted, I performed a recursive search across relevant file types to locate any references to the string HTB. This search was limited to text, configuration, XML, and PHP files. 

grep --include=*.{txt,conf,xml,php} -rnw '.' -e 'HTB' 2>/dev/null
